Deletes a message buffer that was previously created using a call to \mbox{\hyperlink{message__buffer_8h_a2959cd0e3d2bd20d46908e5c9872be36}{x\+Message\+Buffer\+Create()}} or \mbox{\hyperlink{message__buffer_8h_acc84f4a15dbd39ecbad35f7337e78b2c}{x\+Message\+Buffer\+Create\+Static()}}. A message buffer handle must not be used after the message buffer has been deleted.
